# AI Psychologist Admin Panel

Полнофункциональная админ-панель для управления пользователями и сообщениями AI-психолога.

## 🎨 Скриншоты

- Минималистичный чёрно-белый интерфейс
- Управление пользователями в реальном времени
- Просмотр истории диалогов
- Современный дизайн с Tailwind CSS

## 🚀 Технологии

- **Vue 3** - фреймворк для UI
- **Tailwind CSS** - стилизация
- **Axios** - HTTP клиент
- **Vite** - сборщик проекта

## 📦 Установка

### Требования
- Node.js 16+
- npm или yarn

### Шаги установки

```bash
# 1. Клонирование репозитория
git clone <your-repo-url>
cd admin-panel

# 2. Установка зависимостей
npm install

# 3. Запуск в режиме разработки
npm run dev

# 4. Сборка для продакшена
npm run build

# 5. Просмотр собранного проекта
npm run preview
```

Приложение откроется на **http://localhost:3000**

## 🔌 API Endpoints

**Base URL:** `https://ai-psiholog-bot-production.up.railway.app/api`

### Users
```
GET    /users              - Все пользователи
GET    /users/premium      - Premium пользователи
GET    /users/{tg_id}      - Пользователь по ID
POST   /users              - Создать пользователя
PATCH  /users/{tg_id}      - Изменить premium статус
DELETE /users/{tg_id}      - Удалить пользователя
```

### Messages
```
GET    /messages/{tg_id}   - История сообщений
DELETE /messages/{tg_id}   - Удалить историю
```

## 🔐 Аутентификация

При первом входе введи пароль: **admin**

Пароль хранится в localStorage для текущей сессии.

**Для смены пароля:** отредактируй значение в `src/App.vue`

```javascript
const DEFAULT_PASSWORD = 'admin'
```
